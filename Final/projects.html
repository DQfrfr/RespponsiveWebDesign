Smart Home Energy Monitor System

Built with C++, Systems Programming, and IoT concepts

What it does

A program that monitors how much electricity different things in a house use. It helps you see which appliances use the most power.

Main Features

  • Track power usage in real time
  • Monitor different appliances
  • Show usage over time
  • Alert when something uses too much power
  • Calculate electricity costs

How I built it

Written in C++ to make it run fast. The program reads data from sensors and processes it quickly to show current power usage.

What I learned

This project taught me more about C++ and how to work with real-time data. I also learned about memory management and making programs run efficiently.
